# commands
- a - append
- A - append at end of line
- i - insert
- I - insert at beginning of line
- e - jump to end of word
- w - jump to beginning of word
- b - jump to beginning of word
- Ctrl + D - scroll downward
- Ctrl + U - scroll upward

# vscode config
example by chris@machine
https://www.youtube.com/watch?v=g4dXZ0RQWdw&list=LL&index=2

- space + e - toggle explorer
	- right or l - open file
	- r - rename
	- a - create file
	- A - create directory
- tab to cycle between buffers
- Shift + k - hover menu
- searching
	- s+characters to search
	- easymotion plugin in vscode 
	- / - forward search
		- move with n and N
- Ctrl + h or l - move between splits
- : - vim command mode
- Ctrl + j and k - move between horizontal windows
- Ctrl + h and l - move between vertical windows
- space - j - git commands
- space + f - search and replace
- space + v - split vertically
- space - t terminal
