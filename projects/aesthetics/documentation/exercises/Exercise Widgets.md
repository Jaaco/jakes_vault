## Exercise List

Bottom Sheet Pop Up
list view of all exercises, sorted alphabetically
has a text field above for quick filtering
other filters can be added in the future as well
filters could be safed to shared preferences

## Muscle Group List

Bottom Sheet Pop Up
alternative to Exercise List
shows a list of all muscle groups

## Exercises Of Muscle Group

Bottom Sheet Pop Up
can be accessed from Muscle Group List
shows all exercises of selecteid muscle group
this can further be sorted by equipment

## Create or Edit Exercise

Bottom Sheet Pop Up
holds all forms needed to create an exercise
very large (>80 of screen height because)
- dismissing scraps all changes, bigger makes it harder to dismiss by accident
- text field can't be blocked by on screen keyboard

### Select Muscle Groups for Exercise

for this purpose the Muscle Group List can be reused with a 'select multiple' and 'initial selected' parameter
