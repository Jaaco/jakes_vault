# Purpose

this is where the user will spend most of their time (estimated 90%)
this is one of the 5 tabs
the active session is used to track a workout, while you are doing it
it has to be
- intuitive
- quick
- including all necessary functionality
- linked to calendar to view old sessions or session exercises
- able give a quick overview over what was already done
- able to import workouts
- able to adjust planned sets in workout (in case of plan)

# UI

## Active Session

- exercises that were done / planned
- time since start
- reorder exercises
- add new exerice
- add and delete sets
- superset exercises
- modalities to sets / exercises
- session exercise history

## Inactive Session

what is shown when there is no session being tracked right now
- start session buttons
- some overview statistics (duplicates that can be found at other spots as well)
- graphic or animation to fill up empty space?
